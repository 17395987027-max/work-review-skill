# work-review

> 任务复盘 Skill —— 自动从重复失败中提炼教训，精准写入对应位置。

## 场景

Agent 执行任务时反复失败、多次换方案、用户持续不满意。这个 Skill 自动复盘，**定位问题来源**——是某个 Skill 的缺陷，还是通用问题——然后写回正确的位置。

- **Skill 的问题** → 优化该 Skill 的 SKILL.md
- **通用问题** → 写入 CLAUDE.md 核心陷阱表

防止 Agent 下次踩同一个坑，同时让整个技能体系持续迭代。

## 触发条件

- 同一任务反复 ≥ 2 次，用户仍不满意
- 同一问题尝试 ≥ 3 种方案才解决
- 用户明确表达不满（"太慢了""又错了"等）

## 写入策略

| 规则 | 说明 |
|------|------|
| Skill 问题写 Skill | 定位到具体 Skill，优化其 SKILL.md |
| 通用问题写 CLAUDE.md | 跨 Skill 的教训写入核心陷阱表 |
| 每次最多 3 行 | 1 行通用 + 2 行 Skill 专属 |
| 能合就不增 | 已有记录则更新原行 |
| 用户确认 + 告知 | 写入前后均告知用户，不知情不动 |

## 安装

发给 Agent：

> 安装 work-review-skill

或终端：

```bash
npx skills add 17395987027-max/work-review-skill
```
